Environments

Environments describe how the scene is rendered, for example the color of the lighting, or the position of the sun. To interactively adjust the values of the environment you can use the editor, launched using: Procedural.environmentEditor

Members

(static) environments

Sample preset environments for engine
Example
var environment = Procedural.environments[0];

Methods

(static) setEnvironments(environment)

Update the engines environment to the environment configuration passed.
Example
var environment = {
  title: 'custom',
  uniforms: {
    inclination: 0.6,
    fogDropoff: 0.0002
  }
};
